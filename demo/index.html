htest playground

A tiny, self-contained app to run htest against — no server, no network. The examples/tour.yaml manifest drives every page here.

What each page exercises

  • Popup — a button opens a modal that's added to / removed from the DOM (present → absent).
  • Spinner — a loader appears, then clears after a delay (wait_for … until: absent).
  • Form — text, checkbox, radio, textarea, and a file picker, echoed into a summary.
  • Snake — a grid of checkboxes; toggle the snake path to win.
  • Scroll — a tall page with a button far below the fold.

Open this over file://. The tour points at this folder with a relative BASE_URL (file://../demo), so no path editing is needed.
